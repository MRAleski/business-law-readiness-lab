# BUSN 201 Business Law Readiness Lab

This repository contains the GitHub Pages files for the Business Law Readiness Lab.

## Files
- `index.html`
- `style.css`
- `script.js`
- `terms.js`
- `README.md`

## GitHub Pages
Use:
- Source: Deploy from a branch
- Branch: main
- Folder: /root

## Important
The homepage is `index.html`. If your live site displays this README instead of the app, the wrong content was uploaded into `index.html`.

## Time-on-task tracking
Time on task is stored locally by student sign-on name in browser localStorage and prints on the certificate.

## Instructor report
Instructor report is local to the current browser/device because GitHub Pages has no backend database.
