# Business Law Final Exam Prep Lab ## What this is A ready-to-deploy interactive Business Law 201 final exam preparation tool for community college students. The tool uses: - Retrieval practice - Immediate feedback - Error correction - Repetition - Category-based weak-area tracking - A final readiness challenge - A printable certificate No actual exam questions are included. The practice items are concept-based. ## Files Upload all of these files together: - index.html - style.css - script.js - terms.js - README.md ## GitHub Pages Upload Instructions 1. Go to GitHub. 2. Create a new repository. 3. Name it something simple, such as `business-law-final-prep`. 4. Upload all five files into the root of the repository. 5. Go to **Settings**. 6. Select **Pages**. 7. Under **Build and deployment**, select: - Source: Deploy from a branch - Branch: main - Folder: /root 8. Click **Save**. 9. GitHub will provide a public link. Typical link format: `https://yourusername.github.io/business-law-final-prep/` ## Canvas Integration Option 1: Link 1. In Canvas, open the module. 2. Click the plus button. 3. Select **External URL**. 4. Paste the GitHub Pages link. 5. Check **Load in a new tab**. 6. Name it: `Launch Business Law Final Prep Tool`. ## Canvas Integration Option 2: Embed Use a Canvas page and paste this into the HTML editor: ```html ``` ## Recommended Student Instructions Use the tool more than once. Your goal is not to memorize isolated words. Your goal is to recognize legal concepts, correct common mistakes, and apply the rule to a short business scenario. Suggested completion target: - Complete at least two practice modes - Complete the Readiness Challenge - Aim for 80% or higher - Print or screenshot the certificate if required by the instructor ## Instructor Notes The tool stores attempts, best score, mastered terms, and category performance in the browser's local storage. This means progress is saved on the same device and browser, but it does not send student data to Canvas or any external server.
